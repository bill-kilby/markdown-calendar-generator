Creates the markdown file for the current day.

create_markdown.create_markdown() → None¶

Creates the project’s markdown and folders in the current directories.

Creates the root folder, and places a blank year file in it which will be used for linking the months. It then loops through every month and generates the month files, directories, and calls for the days to be made.
